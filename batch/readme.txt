Ces fichiers permettent de recréer des systèmes de batch qui me semblaient intéressants, donc que nos algos arrivent à faire avec une consommation d'énergie normale (Dumb > SingleFrequency > SmallestPath).

Le programme ne lit que le début des fichiers, donc si vous avez des commentaires vous pouvez les mettre directement dedans (à la suite).

Pour les essayer, récupérer la dernière version du code via SVN, compilez-la, puis lancez :
	java Main path/to/file.txt
